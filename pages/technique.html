Roblox Development - Soufyan

1. HTML

De website maakt gebruik van verschillende HTML-elementen om een gestructureerde en overzichtelijke indeling te creëren.

<header>: Bevat de titel van de website en de navigatiebalk, zodat gebruikers gemakkelijk tussen pagina's kunnen wisselen.

<nav>: Binnen de header bevat dit element een reeks <button>-elementen die als navigatieknoppen fungeren.

<footer>: Hierin zijn de sociale media-knoppen (Roblox en Discord) geplaatst, zodat gebruikers eenvoudig naar externe platforms kunnen navigeren.

<img>: Achtergrondafbeeldingen en social media-logo’s worden weergegeven met dit element.

<button>: Wordt gebruikt voor de navigatie-opties, zodat gebruikers door de website kunnen klikken.

Deze structuur zorgt ervoor dat de inhoud logisch is opgebouwd en goed leesbaar blijft.

2. CSS

Voor de styling van de website is CSS gebruikt om een visueel aantrekkelijke en dynamische ervaring te bieden.

Flexbox: Gebruikt in de navigatie en footer voor een nette uitlijning van elementen.

Media Queries: Zorgt ervoor dat de website responsief blijft en goed werkt op verschillende schermformaten.

Kleuren & Achtergrond: Er is een achtergrondafbeelding toegevoegd voor een professionelere uitstraling.

Animaties: Overgangen en effecten worden toegepast om de gebruikerservaring vloeiender te maken.

Deze CSS-technieken maken de website visueel aantrekkelijk en zorgen voor een prettige gebruikerservaring.

3. JavaScript

JavaScript wordt gebruikt om dynamische interacties toe te voegen aan de website.

GSAP Animaties: De GSAP-bibliotheek is geïntegreerd om soepele animaties toe te passen.

Event Listeners: Gebruikt voor de navigatieknoppen en social media links, zodat ze reageren op gebruikersacties.

Onclick-functionaliteit: Knoppen veranderen van pagina bij een klik, wat zorgt voor een interactieve gebruikerservaring.

Door JavaScript wordt de website interactiever en aantrekkelijker voor gebruikers.
